# План разработки Tensu 2.0 (Next.js)

**Цель:** Переписать корпоративный сайт tensu.kz с нуля на Next.js (App Router), используя премиальный дизайн в стиле "Dark Bento Box".

## Этапы реализации:
1. **[Выполнено] Инициализация проекта:** Развертывание Next.js + Tailwind CSS, перенос ассетов.
2. **[Выполнено] Настройка дизайн-системы:** Глобальные переменные в `globals.css` (глубокий темный фон, стеклянные карточки, неоновые свечения).
3. **[Выполнено] Hero Секция:** Сетка Bento (12 колонок), текстовый блок слева, фичи и мокапы справа.
4. **[Выполнено] Секция "Проблема":** Блоки с красным акцентом, показывающие рутину и хаос старых подходов.
5. **[Выполнено] Секция "Решение":** Шахматная сетка стеклянных карточек с синим свечением.
6. **[В очереди] Desktop CRM Demo:** Центральная большая секция с демонстрацией десктопной админки.
7. **[В очереди] Секция "Возможности" (Табы):** Переключатель между "Для учеников" и "Для бизнеса" (здесь покажем скриншоты новой CRM).
8. **[В очереди] Секция "Почему Tensu":** Асимметричная сетка преимуществ.
9. **[В очереди] Секция "Как это работает":** Три шага с неоновыми соединительными линиями.
10. **[В очереди] Секция "FAQ" и "Финальный CTA".**
