# История изменений (Changelog)

## [0.1.0] - Текущая итерация
### Добавлено
- Создан новый проект на Next.js в папке `tensu-new`.
- Настроен Tailwind CSS v4 с пользовательскими `theme` переменными в `globals.css` для создания стиля Dark Bento Box (переменные `--card`, `--glow-primary`, `.tensu-card`).
- Установлена библиотека `lucide-react` для иконок.
- Создан компонент `Navigation` (липкая шапка, блюр при скролле).
- Реализована секция **Hero (Главный экран)** с использованием 12-колоночной сетки Bento Grid и анимациями `animate-float`.
- Реализована секция **Проблема** (акценты красного цвета, предупреждения, список болей клиентов).
- Реализована секция **Решение** (шахматная сетка - staggered grid, сине-фиолетовые свечения).
- Реализована секция **Desktop CRM Demo** (интерактивный симулятор CRM панели).
- Реализована секция **Возможности (Табы)** (плавное переключение между фичами для бизнеса и учеников с использованием клиентских компонентов Next.js).
- Реализованы финальные секции: **Почему Tensu**, **Как это работает**, **FAQ** (интерактивный аккордеон) и **Финальный CTA**.

### Изменено
- `layout.tsx` принудительно переведен в `class="dark"` для корректной работы стеклянных карточек.

### Запланировано
- Все задачи текущего плана выполнены. Готово к деплою!
